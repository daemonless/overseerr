## Deployment

### Podman Compose

```yaml
services:
  overseerr:
    image: ghcr.io/daemonless/overseerr:latest
    container_name: overseerr
    environment:
      - PUID=1000
      - PGID=1000
      - TZ=UTC
    volumes:
      - /path/to/containers/overseerr:/config
    ports:
      - 5055:5055
    restart: unless-stopped
```
